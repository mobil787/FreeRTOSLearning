# FreeRTOSLearning
FreeRTOS Study

**裸机编程：**
+ 指针的指针
+ 状态机编程
+ 面向对象的思想：多态，继承，复用（抽象）
+ 高内聚，低耦合
+ 数据结构，链表

**RTOS:**
+ 交互组件
+ 分时，任务优先级
+ 操作系统有丰富的组件(信号量⋯⋯)

**关于任务：** 
1. 多任务特性 
2. 多任务实现，配置（任务控制块，就是描述任务属性的结构体） 
3. 任务堆栈 

创建任务：
1. 功能需求
2. 任务创建API ( xTaskCreate(), xTaskCreateStatic() )
   任务句柄：就是一个指针，指向任务控制块
